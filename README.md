# GRC & Compliance

Projets autour de la gouvernance, la gestion des risques et la conformite reglementaire.
Contexte : reconversion cybersecurite, specialisation GRC — formation AIS RNCP37680 (Jedha Bootcamp).

## Projets

### smsi-automation-lab
SMSI complet pour une PME fictive (TechShop SAS) — deploiement CISO Assistant, scripting Python pour automatiser les livrables ISO 27001:2022, cartographie NIS2 et RGPD.
Stack : Docker, Python, CISO Assistant

### iac-auditor-agent
Agent d'audit Infrastructure as Code — analyse des fichiers Ansible, Terraform et Docker, mapping automatique vers ISO 27001, NIS2 et CIS Controls, generation de rapports HTML.
Stack : Python, Ansible, Terraform

### agent-rgpd-lea
Assistant conversationnel local specialise conformite RGPD — RAG sur sources officielles CNIL, ANSSI et EUR-Lex, 100% offline via Ollama.
Stack : Python, Ollama, ChromaDB, Mistral

### syntaris-compliance-lab
Evaluation de maturite NIST CSF 2.0 pour une entreprise fictive — scoring automatise, visualisations, plan de remediation RGPD et PCI-DSS.
Stack : Python
